Favorite Cart List in E-Commerce

This Module is to add the favorite products to a separate section like wishlist and checkouts with the added favorite products.

Step 1:

In the Shopping Cart page, the Add to favorites button is to add the products in the cart to your favorites list.

Step 2:

On clicking the button, a window pops up to create a favorite or to add the products to the existing favorite.

Step 3:

The created favorites list are shown here, you can delete the favorites list if you want.

Step 4:

The products are shown in the favorites list could be deleted from here.
