SO Quotation Mail Scheduler

Mail scheduler for Sale Orders and Quotation. The orders will be sent based on the timing scheduled already. If the mail is sent the respective quote will be moved to Sent state.

Step 1:

Select an order click action --> schedule mail.

Select multiple orders click action --> schedule mail.

Step 2:

Select the date to send the mail from the wizard.

Step 3:

Click on Schedule mails menu.

Step 4:

In the Schedule mails menu we can view the scheduled mails.

Step 5:

If needed click on Send now button to send mail immediately.
